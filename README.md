
## Musical Cryptogram

This is a simple script which generates a [Musical
Cryptogram](https://en.wikipedia.org/wiki/Musical_cryptogram) based on a
few provided scales.

#### Usage

```shell
$ ./cryptogram
Usage: scales <scale> <text>
  Available scales:
  0) A♭ major
  1) A♭ minor
  2) A minor
  3) C major
  4) C minor
  5) C♭ major
  6) E minor
  7) E♭ major
  8) E♭ minor
  9) F major
  10) F minor
  11) F♯ major
  12) G♭ major
```

#### Examples

```shell
$ ./cryptogram 1 hail satan
A♭   A♭   B♭   E♭   E♭   A♭   F♭   A♭   G♭
```
